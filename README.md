# 개인 블로그

나 혼자 쓰기 위해 만든 심플하고 모던한 개인 블로그 프로젝트

## 🔨기술 스택
- **Framework:** Next.js 15 (App Router)
- **Styling:** Tailwind CSS
- **Database:** Supabase (PostgreSQL)
- **Auth**: otpauth
- **Editor:**:
  - Tiptap (Headless Editor)
  - Konva (Headless Canvas)
- **Deployment:** Vercel
